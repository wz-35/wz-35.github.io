​ 一直不太理解数组，今天对数组做题个总结。

数组：将一组数据存储到一个指定容器中。用变量指向改容器，可以通过变量一次性得到该容器中的所有数据。

数组的定义：

1.定义数组常用array关键字。语法为：

​ $变量名=array(元素1,元素2,元素3);

我们可以用var_dump()函数来输出数组的元素数，元素的数据类型，元素的值。

例如这个数组，array(3)代表了他的元素有三个，[0]=> string(1) “1”说明了键值为零的元素数据类型为string类 型，值为1。再说一下键值，每个数据都有一个键值，而系统键值是从0开始的。也就是说，第一个元素的键值为 0，第二个元素的键值为1，以此类推。

2.定义变量还可以用[]包裹数据来表示，语法为：$变量名=[元素1,元素2,元素3]；

所以上一个数组也可以写为这样。

3.隐形定义数组，语法为：$变量名[键值]=值1；这种定义法好像只能给我们所定义的键值的元素赋值。

数组的维度：

1.一维数组，就是上文所说的数组。

2.二维数组，数组中所有的元素都是一维数组。

<?php
$arr1=array(
    array('name'=>'wz','age'=>19),
    array('name'=>'xz','age'=>18),
    array('name'=>'yf','age'=>17)
);
var_dump($arr1);
?>

array(3) {
[0]=>
array(2) {
["name"]=>
string(2) "wz"
["age"]=>
int(19)
}
[1]=>
array(2) {
["name"]=>
string(2) "xz"
["age"]=>
int(18)
}
[2]=>
array(2) {
["name"]=>
string(2) "yf"
["age"]=>
int(17)
}
}

3.多维数组，类似于二维数组。

数组遍历：

使数组中所有的数据依次输出出来，使用一些简化的规则来实现自动获取下标以及输出数组元素。

1.foreach遍历数组，语法为：foreach($变量名 as $k=>$v )

<?php
$arr1=array('1',2,"hello");
foreach($arr1 as $k=>$v)
{
    echo "$k\n","$v\n";
}

?>

输出结果为：

0 1 2

1 2 hello

所以k为键值，v为变量值。(其中，k,v的可随意选取)。

其他的不常用，不说了。
