python 中字符串的分割主要使用 split () 函数，其用法为：

定义字符串 a: a=‘1234567890’

然后进行分割： a1=a.split (‘2’)

输出： print (a1)

结果为：

可以看到，.split () 函数中的值为’2’, 也就是被切割的值为 2，字符串 a 就会把 2 的值切去。这个方法可以去除字符串中所有的某一个值。

我们让 a 的值为 121212121 验证一下。

可以看到所有的 2 都被切去了。我们可以往 split 函数里加参数来控制切除的数值的个数。格式为：.split (‘a’,n) 其中 n 为参数。我们看一下效果。

n=1 时

可以看到第一个 2 被切除了，但后面的依旧保留。

然后是字符串的拼接。字符串的拼接用到 join 函数。用法为 'b'.join(a1) （其中 b 是分割符，可以为 -，字母，数字等，也可以不设置。）

我们在上述代码中加入 a2=’’.join (a1) 即可实现对 a1 的拼接。

可以看到 a2 就是 a1 拼接后的结果。

第二种用法，需要引用 os 库。格式为：os.path.join (a1,a2,a3,a4,・・・・・・) 意思是把 a1,a2,a3,a4a・・・・・・等用 \ 链接起来当作路径，我们来浅试一下。

import os
a1='wz'
a2='lk'
a3='fz'
a4='eq'
print(os.path.join(a1,a2,a3,a4))

输出结果为： wz\lk\fz\eq
