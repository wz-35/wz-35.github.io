python中os.listdir的排序问题

2023/5/9

​ os库中的os.path(path)方法可以以表列的方式返回path路径下的所有文件。但是返回的文件顺序是有改变的。例如：

这是原来的文件排序，再看返回的：

可以看到，文件顺序是被打乱的。其实os.listdir()遍历之后的排序是按照文件名的二进制码来排的，我们可以通过改名字来避免乱序。例如将名字命名为000000001.png等。我有个例子：

这个是源文件，可以看到是许多有序号的二维码。我们用os.listdir()给他排序试试。

可以看到仍然是有顺序的。

我们也可以用代码来实现这个效果。

import os
path=os.getcwd() #返回当前进程的工作目录
path_list=os.listdir(path)
path_list.sort(key=lambda x:int(x.split('.')[0]))
print(path_list)

我们先说一下lambda。在python中，lambda函数的语法是唯一的。lambda是python预留的关键字。其语法为：

lambda x,y:x*y(其中x,y是参数，不是必须有两个)

lambda函数可以将函数值赋值给一个变量。如：key=lambda x,y:x*y

就像在上述代码中：path_list.sort(key=lambda x:int(x.split('.')[0]))

意思是将代码上面运行的结果传参赋值给x，：后面的是lambda函数的内容，意思是将我们读取到的文件名从.处切片取序列为1的项，并转化为int类型。然后再作为参数让.sort处理反回结果。

这样也可以使输出结果有序。
