​ os 库中的 os.path (path) 方法可以以表列的方式返回 path 路径下的所有文件。但是返回的文件顺序是有改变的。例如：

这是原来的文件排序，再看返回的：

可以看到，文件顺序是被打乱的。其实 os.listdir () 遍历之后的排序是按照文件名的二进制码来排的，我们可以通过改名字来避免乱序。例如将名字命名为 000000001.png 等。我有个例子：

这个是源文件，可以看到是许多有序号的二维码。我们用 os.listdir () 给他排序试试。

可以看到仍然是有顺序的。

我们也可以用代码来实现这个效果。

import os
path=r"D:\桌面\txt"
path_list=os.listdir(path)
path_list.sort(key=lambda x:int(x.split('.')[0]))
print(path_list)

我们先说一下 lambda。在 python 中，lambda 函数的语法是唯一的。lambda 是 python 预留的关键字。其语法为：

lambda x,y:x*y (其中 x,y 是参数，不是必须有两个)

lambda 函数可以将函数值赋值给一个变量。如： key=lambda x,y:x*y

就像在上述代码中： path_list.sort(key=lambda x:int(x.split('.')[0]))

意思是将代码上面运行的结果传参赋值给 x，：后面的是 lambda 函数的内容，意思是将我们读取到的文件名从。处切片取序列为 1 的项，并转化为 int 类型。然后再作为参数让.sort 处理反回结果。

这样也可以使输出结果有序。
